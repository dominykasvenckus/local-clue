# Local Clue

Local Clue is an application built with React Native and Expo for managing password hints. The app utilizes a local-only storage model to ensure data remains on the device without the use of external servers.

## Features

* **Clue Management:** Create, edit, and delete password hints. Users can favorite specific entries for quick access.
* **Categorization:** Organize hints into predefined categories including Finance, Social, Work, Entertainment, and Shopping.
* **Search and Filter:** Locate specific entries using text-based search or category filtering.
* **Data Portability:** Transfer data between devices using offline QR code generation and scanning.
* **Local Storage:** All data resides exclusively on the device hardware. The application does not utilize cloud synchronization or internet connectivity.
